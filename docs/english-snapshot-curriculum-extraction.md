# English Snapshot Curriculum Extraction

## Purpose
This document pulls out only the curriculum elements that are suitable for an `English Snapshot` assessment.

It is not a summary of all English curriculum content.
It is a reduced extraction for snapshot design.

This should be read together with:

## Snapshot Extraction Rules
Use these rules when deciding what belongs in the snapshot:

- include elements that can be assessed in a short, structured, online format
- include elements that can produce a defensible phase / year best-fit judgement
- include elements that can be scored by clear rules or teacher review
- exclude elements that depend mainly on extended observation or live interaction
- exclude broad oral language performance as a scored online section
- exclude large writing-craft judgments in version 1

## Shared Snapshot-Relevant English Elements
Across both phases, the most snapshot-suitable parts of English are:

- word reading
- reading meaning
- writing

These should remain the three core English snapshot domains.

Across both phases, the least suitable as core online snapshot sections are:

- broad oral language
- extended speaking / presenting
- group discussion performance
- full composition craft

These may still appear as:
- teacher notes
- risk flags
- language/load flags
- teacher probe prompts

## Phase 1 Snapshot-Only Extraction
### Key curriculum elements to keep
Phase 1 should focus on:

- phonemic awareness and phonics
- grapheme-phoneme knowledge
- decoding
- high-frequency words
- early spelling patterns
- vocabulary in context
- sentence meaning
- literal comprehension
- simple inference
- transcription
- sentence construction
- short purposeful writing

### Phase 1 elements to exclude from core snapshot scoring
- broad oral language
- presentation / recitation / performance
- reflective talk as a scored section
- large writing-process judgments
- extended composition craft

### Phase 1 proposed snapshot domains
#### Word Reading
Use this domain to capture:
- letter-sound knowledge
- basic phonics
- decoding
- high-frequency word recognition
- simple spelling pattern transfer

Suggested Phase 1 sections:
- `Word Reading Foundations`
- `Word Reading and High-Frequency Words`

#### Reading Meaning
Use this domain to capture:
- vocabulary in context
- sentence meaning
- literal comprehension
- simple inference

Suggested Phase 1 sections:
- `Vocabulary and Sentence Meaning`
- `Short Text Reading Meaning`

#### Writing
Use this domain to capture:
- spelling / encoding
- sentence construction
- short written response

Suggested Phase 1 sections:
- `Spelling and Encoding`
- `Sentence Writing`
- optional later: `Short Written Response`

### Phase 1 judgement emphasis
In Phase 1, the snapshot should place strongest weight on:

- word reading
- reading meaning
- writing

It should not treat oral language as a scored online strand.

## Phase 2 Snapshot-Only Extraction
### Key curriculum elements to keep
Phase 2 should focus on:

- decoding multi-syllable words
- morphology and word structure
- vocabulary growth and vocabulary in context
- fluent reading of year-level text
- literal and inferential comprehension
- summarising and supported conclusions
- author purpose
- spelling conventions
- sentence control
- paragraph structure
- short purposeful writing

### Phase 2 elements to exclude from core snapshot scoring
- broad discussion behaviours
- collaborative oral interaction as a scored section
- extended presentation skills
- full writing craft and voice judgments
- large multi-draft writing-process judgments

### Phase 2 proposed snapshot domains
#### Word Reading
Use this domain to capture:
- multi-syllable decoding
- morphology
- word analysis
- automatic word recognition

Suggested Phase 2 sections:
- `Advanced Word Reading`
- `Morphology and Word Analysis`

#### Reading Meaning
Use this domain to capture:
- vocabulary in context
- sentence and paragraph meaning
- literal comprehension
- inference
- summarising
- author purpose

Suggested Phase 2 sections:
- `Vocabulary and Meaning`
- `Short Text Comprehension`
- `Inference and Author Purpose`

#### Writing
Use this domain to capture:
- spelling conventions
- sentence variation
- paragraph construction
- short purposeful writing

Suggested Phase 2 sections:
- `Spelling and Conventions`
- `Sentence and Paragraph Writing`
- optional later: `Short Extended Response`

### Phase 2 judgement emphasis
In Phase 2, the snapshot should still produce:
- section judgments
- domain judgments
- an overall phase / year best-fit

But it should not pretend to assess the full complexity of English.

## Recommended Domain Weighting
### Phase 1
- `Word Reading` = `0.40`
- `Reading Meaning` = `0.35`
- `Writing` = `0.25`

### Phase 2
Initial recommendation:
- `Word Reading` = `0.30`
- `Reading Meaning` = `0.40`
- `Writing` = `0.30`

Reason:
- reading meaning becomes more central in Phase 2
- word reading still matters, but should not dominate as strongly as in Phase 1

## Reporting Implications
The report should always include both:

- an overall English snapshot judgement
- a domain / section profile

The report should not reduce English to one opaque score.

The report should foreground:
- overall phase / year best-fit
- confidence
- uneven profile flags
- section summaries
- questions that caused difficulty
- language/load flags where relevant

## Build Implications
Any coder building the English snapshot should use this document as the curriculum filter.

Meaning:
- do not try to encode all curriculum elements
- do not invent oral-language scoring just because oral language is prominent in the curriculum pages
- prioritise what can be assessed online, scored defensibly, and reported clearly
